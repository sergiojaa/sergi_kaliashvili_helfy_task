# Task Manager App

Full-stack Task Manager built with React, TypeScript, and Express.js.

## Tech Stack

- Frontend: React, TypeScript, Vite
- Backend: Express.js, Node.js, TypeScript

## Backend Setup

```bash
cd backend
npm install
npm run dev

Runs on: http://localhost:4000

Frontend Setup
cd frontend
npm install
npm run dev

Runs on: http://localhost:5173

API Endpoints

GET /api/tasks

POST /api/tasks

PUT /api/tasks/:id

DELETE /api/tasks/:id

PATCH /api/tasks/:id/toggle
